About the game

What is this game?

This is a game where you go through 3 different challeges that have you answer questions. The first one is a math quiz, the second is riddles, and the last one is a video searching puzzle.

Why was the game made?

This game is my final project for the course Intro to Website Development. The porject required aspects like using JSON files for storing data, validating inputs in JS, and having everyting look nice with CSS.
